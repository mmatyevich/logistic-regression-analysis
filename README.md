## Overview
This project applies logistic regression to analyse factors associated with binary outcomes in applied datasets.

## Methods
- Logistic regression
- Model interpretation and coefficient analysis
- Original dataset was not available, so a synthetic dataset was created based on reported transmission success rates for each collar for demonstration

## Key Insights
- Identified significant predictors influencing outcome probabilities
- Interpreted odds ratios to understand the direction and strength of relationships
- Demonstrated how model outputs translate into meaningful conclusions in applied contexts

## Tools
R (glm, tidyverse)

## Relevance
Demonstrates application of logistic regression to analyse binary outcomes and support interpretation of results in research and health-related settings
